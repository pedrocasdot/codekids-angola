CodeKids-Angola

A CodeKids-Angola é mais do que apenas uma escola de programação para crianças; somos uma comunidade apaixonada e comprometida em capacitar a próxima geração com habilidades tecnológicas essenciais. Nosso objetivo é proporcionar uma educação enriquecedora e divertida, onde as crianças aprendem a linguagem da tecnologia de maneira criativa e estimulante. Através de nossos cursos dinâmicos e instrutores dedicados, estamos moldando mentes curiosas e preparando-as para um futuro digital.

Nossos Cursos

Introdução à Programação com Scratch

É um curso que ensina os fundamentos da programação de forma fácil e divertida. Utilizando o Scratch, uma plataforma desenvolvida pelo MIT, os alunos aprendem a criar programas e jogos arrastando e soltando blocos de código. O curso foca em conceitos como sequenciamento de instruções, loops, condicionais e variáveis, enquanto estimula a criatividade e o pensamento lógico. É uma ótima maneira para iniciantes de todas as idades começarem a entender como a programação funciona.

Desenvolvimento de Jogos com Roblox Studio

Desenvolvimento de jogos com Roblox Studio é uma maneira divertida e criativa de criar seus próprios jogos interativos e compartilhá-los com amigos e comunidades online. Roblox Studio é uma plataforma de desenvolvimento de jogos que permite que qualquer pessoa, mesmo sem experiência prévia em programação, crie jogos usando uma interface intuitiva e ferramentas poderosas.

Python para iniciantes

Python é uma linguagem de programação versátil e acessível, perfeita para crianças que estão interessadas em aprender a codificar. Com uma sintaxe simples e fácil de entender, Python permite que as crianças mergulhem no mundo da programação de forma divertida e educativa.
